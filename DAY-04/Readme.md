# Java Solutions – Searching Problems

This repository contains my Java practice solutions for search-related problems, all written and tested in **IntelliJ IDEA**.

---

## Implemented Problems

- **LinearSearch** – Iterates through an array to find the target element.
- **SearchIn2DArray** – Finds the target element in a 2D matrix and returns its row/column index.
- **SearchInString** – Checks if a given character is present in a string using `toCharArray()`.
- **SearchInRange** – Searches for a target within a specified index range of an array.
- **FindMin** – Returns the smallest element in an array.
